# 🊁 LeapAI
### *"Your leap into AI starts here."*

> **LeapAI** is the United States' first national lifelong AI learning and workforce transition platform.
> Free. Open-source. Built for every American, age 5 to 120.
>
> Built on [OpenMAIC](https://github.com/THU-MAIC/OpenMAIC) (Tsinghua University, 2026).
> Forked and extended by [AfricanArchitect](https://github.com/AfricanArchitect).

---

## What Is LeapAI?

LeapAI adds the following to the OpenMAIC classroom engine:

- 📚 **20 prebuilt curriculum lessons** (4 pillars x 5 age bands)
- 👦👧 **Age-assurance and parental controls** (COPPA-compliant; Framework S1)
- 👩‍💻 **Privacy-first analytics** (Umami no-cookie)
- 🪪 **LeapAI Learning Passport** (portable digital credential wallet)
- 💰 **Lifelong Learning Account integration**
- 🇺🇸 **Full White House National AI Policy Framework alignment (March 2026)**

## Quick Start

### 1-Click Deploy (Vercel)
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/AfricanArchitect/OpenMAIC)

### Self-Hosted
```bash
git clone https://github.com/AfricanArchitect/OpenMAIC.git
cd OpenMAIC
cp .env.example .env.local
npm install
npm run dev
```
